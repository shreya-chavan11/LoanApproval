# Loan Approval Prediction System

## Project Idea

This project aims to predict whether a loan will be approved or rejected based on user details such as income, credit score, loan amount, and other factors using Machine Learning.

## Objective

To help financial institutions automate the loan approval process, improve decision-making, and reduce financial risk.

## Features

* Accepts user input for loan-related details
* Predicts loan approval status using a trained ML model
* Achieves accuracy of up to **93%**

## Technologies Used

* Python
* Scikit-learn
* NumPy
* Pandas
* Seaborn
* Streamlit
* Label Encoding (for categorical data preprocessing)
* Model: Random Forest Classifier

## How to Run

1. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:

   ```bash
   python project.py
   ```

   or (if using Streamlit):

   ```bash
   streamlit run project.py
   ```
